CORE FEATURES:
JavaScript Modes (js-mode attribute):
  - HTMX: AJAX pagination without page reloads (default)
  - HTMXWithAlpine: Combines HTMX with Alpine.js for reactive state
  - Alpine: Lightweight client-side navigation with Alpine.js
  - PlainJS: Vanilla JavaScript with zero dependencies
  - NoJS: Zero JavaScript - works with forms and standard links

View Types (view-type attribute):
  - TailwindAndDaisy: Full DaisyUI components (btn, join, badge, select)
  - Tailwind: Pure Tailwind utilities without DaisyUI dependency
  - Bootstrap: Bootstrap 5 pagination styles
  - Plain: Custom embedded CSS
  - NoJS: Zero-JavaScript with form-based navigation
  - Custom: Use your own Razor views

TagHelper Components:
  - Pager TagHelper: Full pagination with navigation, page size, and summary
  - PageSize TagHelper: Standalone page size selector
  - SortableHeader TagHelper: Sortable column headers with flip indicators
  - ContinuationPager TagHelper: Token-based pagination for NoSQL databases

Localization:
  - 8 built-in languages: English, French, German, Spanish, Italian, Portuguese, Japanese, Chinese (Simplified)
  - Custom text override for any property
  - Summary templates with placeholders
  - Resource file support for additional languages

CONTINUATION PAGER (NoSQL Support):
  - Token-based pagination for Cosmos DB, DynamoDB, Azure Table Storage
  - Numbered page navigation with clickable visited pages
  - Token history management with configurable limit (default: 20 pages)
  - Automatic query parameter preservation for token validity
  - Support for all JavaScript modes
  - Forward/backward navigation using continuation tokens
  - preserve-query-parameters attribute (default: true)
  - max-history-pages attribute (default: 20)
  - enable-token-accumulation attribute for controlling history storage
